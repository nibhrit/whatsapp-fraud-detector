How it works

Three steps. One tap. Zero behaviour change.

The key product insight: native integration means users never leave the conversation.

1

You receive a message

A suspicious WhatsApp message arrives — a KYC alert, a job offer, a prize claim. It looks real. You're not sure.

2

Tap the AI Shield

A single tap activates analysis on that message. No switching apps. No copying text. The analysis runs inline, inside the chat.

3

Instant verdict, in your language

Fraud or safe — with a plain-language explanation and a clear action to take. Delivered in English, Hindi, or Hinglish based on how you wrote.

Tech stack

Built for accuracy, not complexity.

Every layer chosen deliberately for a portfolio prototype that still behaves like real production logic.

🤖
Claude Haiku
LLM
Anthropic's fastest model. Handles English, Hindi, and Hinglish natively. Structured JSON output for every verdict.
🗄️
ChromaDB
Vector Database
Stores 21 fraud patterns as semantic vectors. Retrieves the 3 most relevant patterns per message in milliseconds.
🔗
RAG Pipeline
Retrieval + Generation
New fraud pattern? Add it to the library. The model adapts instantly — no retraining, no deployment.
⚡
FastAPI
Backend API
Single /analyse endpoint. Pydantic-validated inputs, CORS-enabled, deployed on Render free tier.

Evaluated against 30 real-world messages

15 fraud + 15 legitimate · English, Hindi, and Hinglish · Designed to catch false positives

Live demo

This is what native WhatsApp integration would look like.

The AI Shield is not a separate app. It lives inside the conversation. Tap the shield on any message to analyse it. Or paste a message from your own inbox at the bottom.

The detector responds in the language of the message — automatically. No settings, no selection.

💡 How to use Tap the 🛡 Analyse button on any incoming message. Or type / paste a message at the bottom and press send to add it to the chat.
⚙️ Backend required Make sure the backend is running locally:
cd backend && python3 -m uvicorn main:app
